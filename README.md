# AuraAI

AuraAI is an offline-first AI assistant for Android built using Kotlin and Jetpack Compose. The application combines modern Android development practices with AI-powered conversations, personalized onboarding, trait-based personality customization, local-first data storage, and a custom animated Aura interface. The project demonstrates MVVM architecture, Room Database, DataStore, WorkManager, Coroutines, Flow, and scalable application design principles.
